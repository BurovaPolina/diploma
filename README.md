# Классификатор ценовых сегментов для маркетплейсов

## Описание проекта

Проект собирает данные с трех площадок (Avito, Wildberries, Citilink) с помощью Selenium и BeautifulSoup, а затем классифицирует товары и объекты недвижимости по ценовым сегментам: Бюджетный, Средний, Премиум, Люкс.

Классификация работает в двух режимах:
- Rule-based (пороги цен и ключевые слова) - работает сразу
- ML (обучение модели Random Forest) - требует предварительного обучения

## Что нужно для работы

- Компьютер с  Windows, macOS или Linux
- Установленный Python версии 3.9 или новее
- Папка с проектом

## Как установить

1. Откройте командную строку (нажмите Win + R, введите cmd, нажмите Enter)

2. Перейдите в папку с проектом. Например, если папка лежит на рабочем столе:
   cd C:\Users\ВашеИмя\Desktop\diploma

3. Установите библиотеки одной командой:
   pip install pandas openpyxl scikit-learn joblib

## Как запустить

В командной строке, находясь в папке с проектом, выполните:

python main_classifier.py

## Что произойдёт

Программа загрузит данные из папок avito, сitilink, wildberries, отнесёт каждый товар к одному из четырёх сегментов и сохранит результат в папку classifier/data/results.

В этой папке появится Excel-файл с названием типа classified_20260525_222453.xlsx. Откройте его. В колонке predicted_segment указан сегмент товара. В колонке confidence указана уверенность (чем ближе к 1, тем точнее).

## Если программа работает не так, как ожидалось

Проблема: Нет данных для классификации
Решение: Убедитесь, что в папках avito, сitilink, wildberries есть JSON-файлы. Если их нет, запустите сначала парсеры для сбора данных.

Проблема: Ошибка "No module named 'pandas'"
Решение: Не установлены библиотеки. Выполните команду pip install pandas openpyxl scikit-learn joblib

Проблема: Программа пишет "rule-based режим"
Решение: Это не ошибка. Модель не обучена, но программа работает на правилах. Чтобы обучить модель, выполните две дополнительные команды:

   python -c "from utils import auto_label_data; auto_label_data()"
   python classifier/train_classifier.py

После этого запустите main_classifier.py снова.

## Где смотреть результат

Папка classifier/data/results
Файл classified_дата_время.xlsx

## Как понять результат

В Excel-файле найдите колонку predicted_segment. Там написано: Бюджетный, Средний, Премиум или Люкс.

Чем дороже товар или объект, тем выше сегмент. Для недвижимости на Avito Люкс начинается от 100 миллионов рублей. Для компьютеров на Citilink Люкс начинается от 300 тысяч рублей. Для компьютеров на Wildberries Люкс начинается от 200 тысяч рублей.

## Что делать, если программа запускается, но не видит мои данные

Положите ваши JSON-файлы с товарами в папки:
- Для Avito: папка avito
- Для Wildberries: папка wildberries
- Для Citilink: папка selenium/electronics-stores

Файлы должны называться так, чтобы в имени была площадка: avito_parsed.json, wildberries_pc.json, citilink_pc.json.
